# Protogon

![Protogon, top side](renders/perspective.png)

The protogon is a protoyping board for the EMF Tildagon and Spaceagon badges.
It can be used to prototype a hexpansion. It features EEPROM (with write protection gate) and qwiic connections.

The idea is based on JakeW's [protoboard hexpansion](https://www.tindie.com/products/jakew/protoboard-hexpansion/).

The board fits a 64k EEPROM. There's a jumper you can short to write protect the EEPROM.
A connector is available to plug in qwiic compatible sensors.
